== Release Date: 2026-01-26

== Description

{product_name} - Cluster API integration in Rancher.

This release brings a number of bug fixes, dependency bumps and a new feature:

* Users can now add an annotation `cluster-api.cattle.io/cluster-description` to a CAPI cluster. Once imported, the annotation value will be shown in the Rancher UI as the description for that cluster.
