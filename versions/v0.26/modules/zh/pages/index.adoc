== 什么是 {product_name}？

Cluster API Extension for Rancher，也称为 {product_name}，是一个 Kubernetes Operator，提供 **Rancher Manager 与 Cluster API (CAPI) 之间的集成**，旨在为 Rancher 带来完整的 CAPI 支持。

使用 {product_name}，您可以：

* **自动将 CAPI 集群导入到 Rancher**，方法是在 CAPI 提供的集群中安装 Rancher 集群代理。
* 实施 **GitOps 原生** 工作流来管理 Rancher 中的 Kubernetes 集群。
* 使用 CAPI Operator 可简化 **CAPI 提供程序的管理**。
