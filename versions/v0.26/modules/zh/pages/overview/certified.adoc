== 认证提供者是什么意思？

认证提供者是由 Turtles 团队的 CI 工作流 **积极测试** 的 CAPI 提供者，以验证它们与 Rancher 的 CAPI 扩展正常工作。然而，由于大多数 CAPI 提供者是由 Kubernetes 开放社区维护的上游项目，**我们无法保证** 在发布新提供者版本时不会引入问题。
