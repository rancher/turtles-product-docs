== O que é o comando {product_name}?

A Extensão da API de Cluster para Rancher, também conhecida como {product_name}, é um Operador Kubernetes que fornece **integração entre o Rancher Manager e a API de Cluster (CAPI)** com o objetivo de trazer suporte total ao CAPI para o Rancher.

Com {product_name}, você pode:

* **Importar automaticamente clusters CAPI para o Rancher**, instalando o Agente de Cluster do Rancher em clusters provisionados pelo CAPI.
* Implementar um fluxo de trabalho **nativo do GitOps** para gerenciar clusters Kubernetes no Rancher.
* **Gestão simplificada de Provedores CAPI** usando o Operador CAPI.
