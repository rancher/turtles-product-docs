== ¿Qué es {product_name}?

La Cluster API Extension for Rancher, también conocida como {product_name}, es un Operador de Kubernetes que proporciona **integración entre Rancher Manager y Cluster API (CAPI)** con el objetivo de llevar el soporte completo de CAPI a Rancher.

Con {product_name}, puedes:

* **Importar automáticamente clústeres CAPI en Rancher**, instalando el Agente de Clúster de Rancher en clústeres provisionados por CAPI.
* Implementar un flujo de trabajo **nativo de GitOps** para gestionar clústeres de Kubernetes en Rancher.
* **Gestión simplificada de Proveedores CAPI** utilizando el Operador CAPI.
