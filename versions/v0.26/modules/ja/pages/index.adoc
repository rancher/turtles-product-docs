== {product_name}とは

RancherのためのCluster API拡張、通称{product_name}は、Kubernetesオペレーターであり、**Rancher ManagerとCluster API (CAPI)の統合**を提供し、Rancherに完全なCAPIサポートをもたらすことを目的としています。

{product_name}を使用して、以下のことができます。

* CAPIでプロビジョニングされたクラスターにRancher Cluster Agentをインストールすることで、**自動的にCAPIクラスターをRancherにインポートする**ことができます。
* RancherでKubernetesクラスターを管理するための**GitOpsネイティブ**ワークフローを実装します。
* CAPIオペレーターを使用して、**簡素化されたCAPIプロバイダーの管理**を行います。

{product_name}を使い始めるのを簡単にするために、基本的な概念から高度な機能まで自然にガイドするようにドキュメントを構成しました。ステップバイステップで進むことをお勧めしますが、特定のトピックにすでに精通している場合は、関連するセクションにスキップしても構いません。
